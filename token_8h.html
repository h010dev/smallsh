Enumerator
TOK_0 

placeholder token type

TOK_CMT 

a comment is identified by '#' at the start of the stream

TOK_CTRL_BG 

a background is identified by '&' at the end of the stream

TOK_CTRL_NEWLINE 

a newline is identified by '\n'

TOK_REDIR_INPUT 

an input redirection is a '<' to redirect file io

TOK_REDIR_OUTPUT 

an output redirection is a '>' to redirect file io

TOK_WORD 

a basic token is any word

TOK_COUNT 

count of tokens to allow iterating over them
