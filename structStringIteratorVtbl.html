◆ peek

char(* StringIteratorVtbl::peek) (StringIterator const *const self, unsigned int offset)

Returns the nth character from the iterators cursor, without advancing to the it.


If offset is equal to 0, then the character at the current cursor is retrieved.


If offset is positive, then the iterator peaks ahead of the cursor.


This function is provided to allow for look-ahead parsing by peeking at the next character in the iterator.

Precondition
It is assumed that the iterator is always within bounds of the string it is iterating. Behavior is undefined if this is not the case.
Parameters
selfpointer to iterator object
offsethow far ahead to peak
Returns
the nth character past the iterators cursor, or EOL if no more characters to seek past

Returns a slice copied from the pointer starting at from and ending at the iterators current position - 1.

Precondition
from must point to a valid address within the iterators internal string. This value should be retrieved by a prior call to StringIterator::next(). This function will perform bounds checking to verify the above is true, and will return NULL if it is not.
