Detailed Description

For parsing and storing tokens.

Author
Mohamed Al-Hussein
Date
25 Jan 2022 This is a base class for parsing strings into categorized tokens. This class is not meant to be used on its own, rather it should be inherited and implemented into more specific token classes.
