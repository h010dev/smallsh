void(* TokenVtbl::take) (Token const *const self, StringIterator const *const iter)

Virtual method definition for 'taking', or storing a string as part of the token's value.


Implementations of this method should define how to parse a string using the @iter iterator in order to retrieve a valid token string to be stored at TokenPrivate::value.
