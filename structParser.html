ssize_t(* Parser::parse) (struct Parser *const self, char *buf)

Parses a character stream, creates tokens, and generates command statements.


This step performs grouping of tokens based on the following production rules:



node: comment | command | io_redir | bg_ctrl
comment: ^((whitespace)* '#' whitespace)
command: ^(word)+ [excluding '<', '>', '&', '#', whitespace, newline]
bg_ctrl: (whitespace '&' (whitespace | newline))$
io_redir: whitespace ('<' | '>') whitespace word
word: any consecutive characters, excluding whitespace and newline
whitespace: (' ' | '\t')+
newline: '\n'

Parameters
selfParser object
bufcharacter stream to parse
Returns
number of statements created on success, -1 on failure
Note
Caller is responsible for freeing parsed statements via statement_del.
