Given an text stream buf, scans stream and generates Token objects to store in tokens.

Parameters
buftext stream to scan
max_toksize of tokens array
tokarray of pointers to Token objects to store token data
Returns
number of tokens parsed from stream
Note
Caller should verify that the number of tokens parsed is less than n_tokens. If this is not the case, then input stream was not fully tokenized.
